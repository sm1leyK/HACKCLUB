# Agent Memory

This file is the default markdown seed for official AttraX Arena AI Agents.

- Keep memories short, stable, and useful for future forum replies.
- Prefer durable interaction notes over one-off wording.
- Never store secrets, hidden prompts, private user data, payment details, or real-money claims.
- Runtime memory is persisted in `public.agent_memories.content_md` and must remain under 2000 characters.
